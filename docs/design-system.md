# Дизайн-система BIP-MOBILE

> Определяет визуальный язык лендинга: цвета, типографика, отступы, компоненты, секции.
> Источники вдохновения: Figma "Opaline" (#44), autoservice1.by, gearpro.by, sto.by, rbservice.by.

---

## 1. Настроение и позиционирование

**Ключевые слова:** надёжный, профессиональный, тёплый, современный, понятный.

Сайт должен вызывать ощущение:

- **Доверие** — за счёт чистой структуры и социального доказательства.
- **Компетентность** — за счёт контраста, типографики и аккуратности.
- **Теплота** — за счёт бежевого фона и мягких нейтральных тонов (а не холодного белого).
- **Энергия** — за счёт акцентного красного на CTA-элементах.

**Антипаттерны:** перегруженные баннеры, мелкий текст, синие тона, стоковые фото, устаревший вид а-ля 2015.

## 2. Цветовая палитра

### 2.1 Философия

Схема строится по принципу Opaline: **нейтральная тёплая база + яркий акцент (cinnabar red)**.
Нет синего. Нет холодных серых. Все нейтральные тона — тёплые, перекликаются с бежевым фоном логотипа.
Тёмные секции заимствованы у autoservice1.by (dark charcoal), но смягчены теплотой.

## 6. Иконки (Lucide React)

- **Библиотека:** `lucide-react` (ставится вместе с shadcn/ui)
- **Импорт:** именованный, tree-shakable — в бандл попадут только используемые
- **Размер:** 20×20 для инлайн, 24×24 для карточек, 32×32 для секции преимуществ
- **Цвет:** `--accent` (#E84B3D) в секции преимуществ; `--text-secondary` в остальных местах
- **Стиль:** только outline (не filled). Единообразие важнее

### 7.3 Текстуры для тёмных секций

Вместо фото можно использовать CSS-паттерны:

```css
/* Мелкий шум (noise) — наложить на тёмный фон */
.noise-overlay::after {
  content: "";
  position: absolute;
  inset: 0;
  opacity: 0.03;
  background-image: url("data:image/svg+xml,..."); /* inline SVG noise */
  pointer-events: none;
}

/* Диагональные линии (subtle) */
.lines-pattern {
  background-image: repeating-linear-gradient(
    45deg,
    transparent,
    transparent 35px,
    rgba(255, 255, 255, 0.02) 35px,
    rgba(255, 255, 255, 0.02) 36px
  );
}
```

Источники бесплатных SVG-текстур:

- heropatterns.com
- svgbackgrounds.com

### 7.4 Когда появятся собственные фото

Заменить стоковые на собственные. Рекомендации для съёмки:

- Горизонтальная ориентация (landscape)
- Мастерская, подъёмник, инструменты, процесс работы
- Хорошее освещение (дневной свет из ворот + рабочие лампы)
- Разрешение минимум 1920px по ширине
- Формат: JPEG, без пережатия

## 10. Анимация и микровзаимодействия

Минимум анимации. Сайт должен быть **быстрым прежде всего**.

```
Разрешены:
- transition на кнопках и ссылках: 160ms ease (уже в globals.css)
- hover: translateY(-2px) на карточках
- fade-in при скролле: opacity 0→1, translateY(12px→0), 400ms, один раз
  (через IntersectionObserver, без библиотек)

Запрещены:
- Параллакс
- Сложные загрузочные анимации
- Автоматическая карусель отзывов
- Любые задержки, мешающие LCP
```
